# Django Blog Website

## 📌 Project Description
This is a Django-based blog web application where users can view posts, read details, and explore categories.

## 🚀 Features
- Blog post listing
- Detail page for each post
- Category-based posts
- Contact form
- Pagination

## 🛠️ Technologies Used
- Python
- Django
- HTML, CSS (Bootstrap)
- MySQL 

## ▶️ How to Run

```bash
git clone https://github.com/autitl22019/blog-website.git
cd blog-website
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
